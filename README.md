# Indian Healthcare Patient Records Analysis

## 📌 Project Overview

This project analyzes 100,000 Indian healthcare patient records to identify disease trends, health indicators, treatment outcomes, hospital performance, insurance coverage, and demographic patterns.

## 🛠️ Tools & Technologies

* Python
* Pandas & NumPy
* Matplotlib & Seaborn
* SQL
* Excel
* Power BI

## 📊 Key Analysis

* Disease distribution and patient demographics
* Blood Glucose, HbA1c, Cholesterol, and BMI analysis
* Treatment outcomes
* Hospital type performance
* Insurance coverage analysis
* Gender and regional healthcare trends

## 🎯 Objective

To analyze healthcare data and generate meaningful insights that can support better understanding of patient health patterns and treatment outcomes.

## 📁 Project Components

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* SQL Analysis
* Power BI Dashboard
* Business Insights
